# 项目脚本说明

本目录包含项目的辅助脚本工具。

## 脚本列表

### detect-shadow-clones.sh

**用途**: 检测项目中的影分身代码（代码重复）和技术债务

**功能**:
1. 检测重复的类定义
2. 检测重复的事件参数类型（EventArgs）
3. 检测重复的 DTO 类型
4. 检测技术债务标记（TODO/FIXME/HACK）
5. 检测配置硬编码问题
6. 检测重复的方法签名
7. 检测 UTC 时间使用（应使用本地时间）

**使用方法**:
```bash
./scripts/detect-shadow-clones.sh
```

**输出**:
- 绿色 ✅: 检查通过
- 黄色 ⚠️: 发现问题，需要关注
- 红色 ❌: 发现严重问题

**注意**: 此脚本仅用于检测和警告，不会阻止 PR 提交。
